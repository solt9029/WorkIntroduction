TOEIC Generator

Description

This is a program that you can make TOEIC style questions by posting images. This program uses Docomo API to detect what are in the image you post. Then, it uses the database which contains a lot of TOEIC questions and offers questions suitable for your image. I made this program because I like making TOEIC questions.
